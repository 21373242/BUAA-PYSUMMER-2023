# BUAA-PYSUMMER-2023

* 项目名：HangEat
* 前后端：PySide6-Fluent-Widgets
* 数据库：MySQL

项目结构

```
.
├── __pycache__
├── README.md
├── resource    // 放置图片等资源
├── src			// 放置 *.py 源代码
└── ui			// 放置 *.ui 文件
```

conda 虚拟环境依赖安装

```
conda create -n hangeat python=3.11
conda activate hangeat
pip install "PySide6-Fluent-Widgets[full]" -i https://pypi.org/simple/
```
